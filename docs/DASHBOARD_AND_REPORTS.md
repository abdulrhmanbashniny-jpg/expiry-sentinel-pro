# مخطط لوحة التحكم والتقارير والتنبيهات

يحتوي هذا المستند على تفاصيل واجهة المستخدم والتقارير المطلوبة للنظام.

## 1. لوحة التحكم (Dashboard)
- **إحصائيات سريعة**: إجمالي التنبيهات، المهام المعلقة، المهام المصعدة.
- **الرسم البياني**: توزيع انتهاء الصلاحية خلال الـ 30 يوم القادمة.
- **قائمة المهام**: عرض المهام حسب الأولوية (موظف، مشرف، مدير).

## 2. التقارير (Reports)
- **تقرير الالتزام**: تتبع سرعة استجابة الموظفين للتنبيهات.
- **تقرير التصعيد**: قائمة بالمعاملات التي وصلت لمستوى المدير أو HR.
- **تقرير المخزون/الوثائق**: قائمة شاملة بجميع العناصر وتواريخ انتهائها.

## 3. التنبيهات (Alerts)
- **تنبيهات فورية**: عبر النظام عند اقتراب موعد الانتهاء.
- **رسائل البريد/الواتساب**: إرسال إشعارات خارجية عند التصعيد.
- **إشعارات المتصفح**: تنبيهات Push للموظفين النشطين.

تم إعداد هذا المخطط بناءً على طلب المستخدم لتوفير تجربة متكاملة للإدارة والمتابعة.
